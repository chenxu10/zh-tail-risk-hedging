## 共同的负面知识
1. 不要使用高斯模型和BSM(管大宇、许哲、戴国晨实证数据证伪)
2. 不要把自己的策略依赖在尾部行情不出现的基础上（广泛地交易多个市场3-5年肯定遇到一次）
3. 期权买方不能错过情绪最极端的时候(分钟到半小时级别)，不能只考虑delta，要考虑Vega，自己实战多次犯了想要卖在最高位的错误(Kevin、管大宇)
4. 大行情不是预测出来的，是不断平掉利润，买虚值期权接力(管大宇)
5. 不要不关注下注比例(不要不使用凯利公式的思维方法)
6. 为什么大多数人包括自己无法执行？赚的是谁的钱？(胜人者有力，自胜者强)
    - 自己喜欢买可能发生的小损失保险而不是极不可能的大损失尾部保险
    - 难以接受自己花的成本买的对冲不被大多数同行接受，但是只有大多数同行不接受，你买到的价格才是性价比高的(十分吸引博客)
    - 与投资人沟通，投资人愿意买单很难(许哲) 
7. 嘉宾的共同思维方法论：
    - 不是去想什么是对的。而是想什么是错的。错了还能不能活，如何避免错误
    - 决策不能依赖于用历史数据做归纳法总结
    - 做对四件事：定标的、动态展期、尾部风险监控、退出
    - 创业就是追求高盈亏比的交易(多元化、交叉、遗传、变异、失败的淘汰、成功的复制、反馈)
    - 不要去防御不确定性，要和不确定性与黑天鹅做朋友，要拥抱它
